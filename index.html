ברוכים הבאים לאתר הזיכרון לעליזה סתרי ז"ל






אודות

סבתי, עליזה סתרי נולדה בתאריך 13.5.1940 בכרמאנשאה, איראן
בת בכורה מתוך 7 אחים לפנחס ורינה ציוני ז"ל.
המשפחה עלתה ארצה בשנת 1950 והתמקמה בבאר שבע.
בשנת 1957 סבתי נישאה לסבי מורד (ז"ל), בעקבות הנישואין עברה להתגורר בכפר סבא.
בשנת 1958 נולדה דודתי שולה וכ 4 שנים לאחר מכן נולד אבי אילן, באותה השנה סבי נפטר והמשפחה חזרה לבאר שבע.
בשנת 1972 החלה לעסוק כסייעת בגן ילדים עד ליציאתה לפנסיה.
לסבתי נולדו כ-6 נכדים ו-5 נינים (נכון להיום).
סבתא עליזה נפטרה באופן פתאומי בשנת 2016 בעקבות מפרצת.
אנו, משפחתה, היינו כל עולמה ועשתה הכל על מנת שנהיה מאושרים.
סבתא עליזה אהבה לסרוג, לחגוג ימי הולדת ואת חג פורים ובעיקר לבלות במועדון.

תשובות

  1. אופי האתרים

    גרסת Web 1.0 התאפיינה בעיקר על ידי עמודי אינטרנט סטטיים שתפקידם העיקרי היה לספק ולפרסם מידע.
    גרסת Web 2.0 דפי אינטרנט יותר דינמיים - כלומר, יכולים להשתנות בזמן אמת. בנוסף, נוספו תכונות לאתרי האינטרנט השונים. כלומר, מעבר להצגת מידע, אתרים יכולים לשמש כאתרי סחר, בלוגים, רשתתות חברתיות וכו'.
    גרסת Web 3.0 מאופיינת על ידי התווספות הבינה המלאכותית ולמידת המכונה כך שהרבה מאתרי האינטרנט מותאמים אישית למשתמש.
    למשל - כניסה לאתר אמזון.
    גרסת Web 1.0 האתר שימש כמאגר מידע עצום של ספרים.
    גרסת Web 2.0 באתר ניתן לרכוש ספרים ובעתיד אף הפתחתח לאופציות של רכישה מתוך מגוון רחב מאוד של מוצרים באופן כללי.
    גרסת Web 3.0 לכל משתמש יוצעו מוצרים שונים לרכישה על פי היסטוריית הרכישות של אותו המשתמש.

  2. פרסום באינטרנט

    גרסת Web 1.0 הפרסום באינטרנט הוא על ידי חברות/אנשים פרטיים שמעוניינים להציג מידע מסויים. גרסת Web 2.0 הפרסום באינטרנט יכול להיות על ידי אנשים שונים שיכולים לשתף מידע ולפרסם תוכן. קיימת אינטרקציה בין אותם האנשים בניגוד לגרסה הקודמת שם הפרסום ברשת היה חד צדדי.
    גרסת Web 3.0 מכונה יכולה להציג למשתמש תוכן מותאם אישית בהתאם לבקשת המתשמש.
    למשל - חיפוש השאילתה "אייפון"
    גרסת Web 1.0 יופיעו תוצאות הקשורות לאייפון - עמוד עסקי של אפל, עמוד פרטי של סטיב ג'ובס.
    גרסת Web 2.0 לתוצאות מגרסה 1 יתווספו תוצאות שמובילות לאופציה של רכישת המכשיר, ייתכנו אתרים בהם ניתן לקרוא ולכתוב חוות דעת על המכשיר.
    גרסת Web 3.0 יופיעו אתרים אפשריים לרכישת המכשיר המתבססים על היסטוריית החיפושים של המשתמש. למשל, אם בעבר משתמש בציע רכישה באתר מסויים סביר להניח שאותו האתר יופיע כאחד מתוצאות החיפוש הראשונות.

  3. תקשורת ב Web

    גרסת Web 1.0 באתרים, הדרך ליצירת קשר הייתה על ידי אימייל/מספר טלפון שהופיע בדף. הכוונה, כדי לבצע התקשרות מסויימת קיים צורך בעוד פעולה והדבר לא יכול להיעשות בצורה ישירה דרך האתר.
    גרסת Web 2.0 ניתן לבצע התקשרות עם גורמים מסויימים על ידי צ'אט עם בן אנוש שמופעל באתר.
    גרסת Web 3.0 בנוסף לתקשורת עם בני אדם כפי שהתאפשר בגרסה 2, בגרסה 3 ניתן יהיה לתקשר עם רובוט בינה מלאכותית.
    למשל - התקשרות עם נציג שירות בבנק.
    גרסת Web 1.0 כניסה לאתר הבנק שם מופיע מספר הטלפון של שירות הלקוחות, חיוג למספר זה וההתקשרות מתבצעת על ידי שיחה קולית.
    גרסת Web 2.0 באתר הבנק ניתן להתחבר לחשבון אישי, שם בשעות הפעילות יש מענה אנושי בצ'אט כך שאותו נציג יודע עם מי הוא משוחח ויכול לראות את פרטי החשבון של אותו אדם.
    גרסת Web 3.0 בגרסה זו, מתווספת האפשרות לשוחח עם נציג שמופעל על ידי בינה מלאכותית בכל שעות היום. ניתן לשאול את הבינה המלאכותית שאלות שהתשובה אליהם תהיה בהתאמה למידע שיש לבינה על אותו אדם.
